# OLMGLASS Website

Static website for `olmglass.com` in Spanish.

## Content

The page presents OLMGLASS as a windows, doors, and design company. It includes:

- ventanas y puertas balconeras
- sistemas correderos Smart-Slide and HST
- persianas exteriores
- mosquiteras
- alféizares and finishing details
- measurement, advice, and installation
- location: Carrer del Malet, 4, 08791 Sant Llorenç d'Hortons, Barcelona
- phone: +34 629 366 540
- separate pages for solutions, sliding systems, and process

The structure is inspired by the service information from `https://robstan.pl/okna/`, but the copy, language, and design are original for OLMGLASS.

## Launch

Open `index.html` in a browser, or upload the whole folder to a static hosting provider.

## Before publishing

Update the email in `index.html` when the final contact email is ready. The phone number, address, and Google Maps link are already included.
